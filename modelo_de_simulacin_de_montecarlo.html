MODELO DE SIMULACIÓN DE MONTECARLO

Gráfico

Paso a Paso

Como tal el modelo puede ser explicado de la siguiente forma:

  1.  Analizamos la mejor distribución de probabilidad para las principales variables.
  2. Obtenemos la distribución de probabilidad acumulada para cada variable.
  3. Se establecen intervalos de números aleatorios para cada variable.
  4. Realizamos una generación de números "aleatorios".
  5. Simulamos el experimento.
